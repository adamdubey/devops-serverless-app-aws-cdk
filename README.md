## QRH Commands

```sh
# Install project dependencies
pip install -r requirements.txt

# List CDK Stacks
cdk ls

# Synthesize CDK Stacks
cdk synth

# Display CDK Stack changes
cdk diff <STACK_NAME> --profile <PROFILE_NAME>

# Deploy CDK Stack
cdk deploy <STACK_NAME> --profile <PROFILE_NAME>
```
